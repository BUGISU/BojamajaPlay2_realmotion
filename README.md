<h2>프로젝트 소개</h2>
<h3>보자마자 PLAY2 리얼모션</h3>
 
모바일로 출시된 보자마자 PLAY2의 모드를 차용한 시스템
Leap 모션을 이용한 모션인식 게임으로 10개의 게임으로 구성되어 있습니다.<br>각 게임은 30초 동안 진행되고 클래식 모드와 랭킹 모드가 존재합니다.<br>
클래식 모드에서는 원하는 게임으로 선택하여 개인의 점수를 기록하고 랭킹 모드는 10개의 게임을 모두 플레이하고 점수를 등록 할 수 있고 100위 안에 드는 사람들은 랭킹 존에서 자신의 점수를 확인 할 수 있습니다.
<br>
보자마자 Play2 모바일 버전에서 리얼모션으로 변형이 가능한 프로젝트 6개와 새로 추가된 게임 4개
